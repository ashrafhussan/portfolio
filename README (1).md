## ⚙️ How the Contact Form Works

1. Visitor fills in the form on the portfolio
2. Form sends a `POST` request to the Node.js/Express backend on Render.com
3. Backend validates the data and sends **two emails** via Nodemailer + Gmail:
   - 📩 Notification email to **me** with the visitor's message + a Reply button
   - 📧 Auto-reply to the **visitor** confirming their message was received
4. Toast notification confirms success on screen

**Backend tech:** Node.js · Express · Nodemailer · express-rate-limit · dotenv
